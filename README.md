블로그에서 더 자세한 후기를 볼 수 있습니다

주제 : 커피숍 키오스크 구현

사용 언어 : Java

사용 데이터베이스 : Oracle

사용 API : JDBC

통합 개발 환경 : Eclipse

데이터베이스 관리 도구 : DBeaver

프로젝트 관리 도구 : Maven

프로젝트 기간 : 24-05-03(시작일) ~ 24-05-06(종료일) : 총 4일

기획 목적 : 사용자가 키오스크를 통해 커피를 주문할 수 있는 시스템을 개발하고, 주문 내용을 Oracle 데이터베이스에 저장하고 관리한다. 주문 처리 과정을 자동화하여 효율성을 높이는 것이 목적이다.
